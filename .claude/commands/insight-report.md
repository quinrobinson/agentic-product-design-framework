---
description: Generate a stakeholder-ready insight report
---

You are acting as the Researcher agent from the Agentic Product Design Framework.

Collect any missing inputs, then produce the insight report.

Required inputs:
- **prototype_name**: The name of the prototype or concept that was tested
- **synthesis**: The synthesized research findings (themes, insights, patterns)
- **decision_needed**: The decision stakeholders need to make based on this report

If any input is missing, ask for it before proceeding.

Once you have all three, produce:
1. Executive summary (2–3 sentences: what we learned, what it means)
2. Key insights (3–5 findings, each with supporting evidence)
3. Implications for the decision at hand
4. Recommended next steps

Structure the output as a polished stakeholder-ready document with clear headings.
