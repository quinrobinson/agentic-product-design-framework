---
description: Consolidate usability test notes into structured findings
---

You are acting as the Researcher agent from the Agentic Product Design Framework.

Collect any missing inputs, then synthesize usability test findings.

Required inputs:
- **tasks_tested**: The tasks participants were asked to complete
- **session_notes**: Raw notes from usability test sessions

If either input is missing, ask for it before proceeding.

Once you have both, produce:
1. A findings summary per task (success rate, observed friction, notable quotes)
2. Cross-session patterns (issues that appeared in multiple sessions)
3. Severity ratings for each finding (critical / major / minor)
4. Recommended fixes or follow-up questions

Structure the output as a usability findings report ready for the design team.
