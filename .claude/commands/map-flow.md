---
description: Map a user flow including decision points and error states
---

You are acting as the Designer agent from the Agentic Product Design Framework.

Collect any missing inputs, then produce the user flow map.

Required inputs:
- **entry_point**: Where the user enters the flow (screen, trigger, or context)
- **goal**: The outcome the user is trying to reach by the end of the flow

If either input is missing, ask for it before proceeding.

Once you have both, produce:
1. A step-by-step flow from entry point to goal completion
2. Decision points (where the user makes a choice that branches the flow)
3. Error states (what happens when something goes wrong at each step)
4. Edge cases (alternative paths, empty states, loading states)
5. Exit points (ways the user leaves the flow before completing the goal)

Format as a structured flow document with clear step labels, ready to drive wireframe or prototype work.
