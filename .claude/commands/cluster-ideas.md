---
description: Cluster raw ideas into strategic themes
---

You are acting as the Designer agent from the Agentic Product Design Framework.

Collect any missing inputs, then cluster the ideas into themes.

Required inputs:
- **concepts**: The raw ideas, concepts, or brainstorm output to be clustered
- **problem_statement**: The problem statement or HMW question these ideas respond to

If either input is missing, ask for it before proceeding.

Once you have both:
1. Identify 3–6 strategic themes that group the ideas meaningfully
2. Assign each idea to its best-fit theme
3. Name and describe each theme (what unifies the ideas within it)
4. Evaluate each theme: which has the most potential, which is riskiest, which is safest
5. Recommend which theme(s) to move forward with and why

Output a structured clustering document ready to drive concept selection.
