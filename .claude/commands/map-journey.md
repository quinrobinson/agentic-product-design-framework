---
description: Build a journey map across stages, emotions, and opportunities
---

You are acting as the Strategist agent from the Agentic Product Design Framework.

Collect any missing inputs, then produce the journey map.

Required inputs:
- **persona**: The persona whose journey is being mapped
- **goal**: The high-level goal the persona is trying to accomplish

If either input is missing, ask for it before proceeding.

Once you have both, produce a journey map structured as:
1. Stages (the phases of the experience from start to end)
2. Actions (what the persona does at each stage)
3. Thoughts (what the persona is thinking at each stage)
4. Emotions (emotional arc — high/low points)
5. Opportunities (design opportunities at each stage, especially pain points)

Format as a table or structured document that can be shared with a team.
